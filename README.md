# transport-electron

一个使用 Electron-Vite 构建的支持跨平台文件传输的项目

## Initialization

首先进入设置界面配置文件上传和下载接口地址，添加后即可正常使用，默认保留的 127.0.0.1 是不能正常使用的

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
